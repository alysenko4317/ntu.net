# Матеріали курсу «Стек технологій .NET»

Цей репозиторій містить супровідні матеріали до лекцій та лабораторних робіт з дисципліни **«Стек технологій .NET»**, які були прочитані в Національному технічному університеті «Харківський політехнічний інститут». Матеріали призначені для допомоги студентам у вивченні ключових концепцій розробки з використанням платформи .NET та мови C#.

---

## Огляд курсу

Курс охоплює фундаментальні та поглиблені теми, необхідні для побудови надійних та масштабованих застосунків на платформі .NET. Розглядаються питання роботи з WinForms, колекціями, LINQ, Entity Framework, багатопотоковістю, ASP.NET, Docker та іншими технологіями.

---

Матеріали лекцій знаходяться в каталозі [`Lections/`](Lections/).

| # | Тема | Опис |
|---|------|------|

| **L13** | [ASP.NET — Конфігурація та стан](Lections/L13%20-%20ASP.NET%20-%20Configuration%20%26%20State/) | Управління конфігурацією застосунку та збереження стану. |
| **L14** | [ASP.NET — Маршрутизація та MVC](Lections/L14%20-%20ASP.NET%20-%20Routing%20%26%20MVC/) | Система маршрутизації, патерн MVC, мінімальні веб-застосунки. |
| **L15** | [Docker](Lections/L15%20-%20Docker/) | Контейнеризація застосунків, робота з Docker, nginx та Redis. |
